テーブル設計

使用する２つのテーブルについて

上がユーザーの名前、投稿内容、ID、投稿日時などをまとめたテーブルです

下が投稿内容の種類を紐付けてあるテーブルです

テーブル名 フィールド 型 説明
SNOWRESQUE
ID INT PIMARY KEY AUTO_INCREMENT
TYPE INT 投稿内容の種類
SOLUTION INT 解決済みなら1,未解決なら2
NAME VARCHAR(100) 投稿者名
TEXT VARCHAR(100) 投稿内容
CREATED DATETIME 更新日時
テーブル名 フィールド 型 説明
RESQUETYPE
TYPE TYPE SNOWRESQUEのTYPEと紐づけ
DISCRIPTION VARCHAR(100) 投稿内容の種類の具体的な内容
